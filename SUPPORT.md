# Support

OpenInstall is pre-1.0 open source software.

## Questions And Help

Use GitHub Discussions if enabled, or open a GitHub issue with the `question`
label.

Please include:

* OpenInstall version.
* Windows version.
* Whether the build is an official signed release or a local/dev build.
* The `openinstall://` link or repo URL if it is safe to share.
* Screenshots or logs when relevant.

## Bugs

Open a bug report using the issue template. Include reproduction steps and
whether the issue happens with a direct `.oip` link, an imported repo, or app
launch/update management.

## Security

Do not report vulnerabilities in public issues. Follow [SECURITY.md](SECURITY.md).
