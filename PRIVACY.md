# Privacy Policy

OpenInstall does not require an account and does not collect telemetry.

## Data Stored Locally

OpenInstall stores local application state on the user's Windows profile, such
as imported repository URLs, installed app metadata, publisher-key pins, cached
blocklist data, and user settings. This data stays on the user's device unless
the user shares it manually.

## Network Requests

OpenInstall makes network requests only to perform user-requested package-manager
work, such as:

* downloading `.oip` packages from selected sources,
* fetching imported OpenInstall repository catalogs,
* refreshing the public OpenInstall blocklist feed,
* opening links selected by the user.

Those requests are sent directly to the relevant host. OpenInstall does not run
a tracking service for this data.

## Crash Reports And Analytics

OpenInstall does not include crash-report upload, analytics, advertising,
tracking pixels, or third-party telemetry.

## Security Reports

Security vulnerabilities must be reported privately through GitHub Security
Advisories:

https://github.com/baar-Software/OpenInstall/security/advisories/new

## Contact

For privacy-related questions, open a non-sensitive GitHub issue:

https://github.com/baar-Software/OpenInstall/issues/new
